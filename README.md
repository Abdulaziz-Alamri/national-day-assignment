# Saudi National Day Quiz App

![Welcome](./assets/images/screenshots/nationalday.png)

## Description

The Saudi National Day Quiz app is a fun and educational application designed to test your knowledge about Saudi Arabia. With a series of engaging multiple-choice questions, users can explore various aspects of Saudi culture, geography, and history. The app features a user-friendly interface with a smooth navigation experience, making it perfect for anyone interested in learning more about Saudi Arabia.

## Features

- Interactive Quiz: Answer multiple-choice questions about Saudi Arabia and test your knowledge.
- Score: The app keeps record of your score as you progress
- Results Screen: View your results and see how well you performed.
- Visual Appeal: Enjoy vibrant images and engaging visual elements that enhance the user experience.
